# INMP441 Microphone Connection to STM32F401RTC6

| INMP441 Pin | STM32F401RTC6 Pin |
|------------|------------------|
| VDD        | 3.3V             |
| GND        | GND              |
| SCK        | PB10 (I2S1_CK)   |
| SD         | PB15 (I2S1_SD)   |
| WS         | PB12 (I2S1_WS)   |
| L/R        | GND (Left)       |
